# Karthik Mohan - Top Web Developer Portfolio

This is a modern, responsive portfolio website for Karthik Mohan, built with HTML5, CSS3, and JavaScript.

## Features

- **Hero Section**: Dynamic background with animated text.
- **About Me**: Professional summary.
- **Skills & Expertise**: Visual representation of technical skills.
- **Projects**: Gallery of featured work.
- **Services**: Overview of professional offerings.
- **Contact**: Interactive contact form.
- **Responsive Design**: Mobile-first approach, looks great on all devices.
- **Dark Mode**: Sleek dark theme with neon accents.

## How to Run

1.  **Clone or Download**: Ensure you have the project files locally.
2.  **Open in Browser**: Simply double-click `index.html` to open it in your default web browser.
3.  **Local Server (Recommended)**: For the best experience, run a local server (e.g., using VS Code "Live Server" extension or Python).

    ```bash
    # using python 3
    python3 -m http.server
    # then open localhost:8000
    ```

## Customization

- **Resume**: Replace the placeholder text in the "About Me" and "Experience" sections with real data.
- **Images**: Add project images to the `assets/` folder and update the `src` attributes in `index.html`.
- **Links**: Update `href` in the contact section with real social media profiles.
